# A personal system to get notified about Chelsea FC matches

## Features

- [ ] Config your preferences
    - [ ] Your email
    - [ ] Your phone number
    - [ ] Your WhatsApp/Telegram/SMS provider
    - [ ] Your preferred notification channels
- [ ] Get notified about next 5 Chelsea FC matches
    - [ ] Through email
    - [ ] Through WhatsApp/Telegram/SMS
    - [ ] In app-notifications

## Technologies

- [ ] Next.js
- [ ] Tailwind CSS
- [ ] Shadcn/UI
- [ ] SQLite
- [ ] Drizzle ORM
- [ ] Vercel
- [ ] Vercel Cron
- [ ] Turso
- [ ] Twilio
- [ ] Resend

- [ ] football-data.org: https://www.football-data.org/documentation/quickstart
